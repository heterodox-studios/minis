# Euro Pallet (1/10 scale) Technical documentation

The product is a small decorative item. It is 3D printed and distributed in a variety of colours.

## Product Details

**Product name**
: Euro pallet 1/10th scale

**Model number**
: N/A

**SKU**
: SKUs take the form “pallet-pla-CC-XX” where CC is colour code and XX is number of pallets in a pack. For example `pallet-pla-wh-01` is a single white pallet.

**Intended use**
: Home decor

**Target user group**
: Adults

**Technical specifications**
: 3D printed from PLA, contains 4 small captive disc magnets.
